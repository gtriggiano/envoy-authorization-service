# Changelog


## [v2.0.0](https://github.com/gtriggiano/envoy-authorization-service/compare/v1.0.3...v2.0.0) - 2025-11-30

### BREAKING CHANGES:

This release introduces a fundamental architectural shift in how the service evaluates
authorization policies. The changes affect configuration format, API contracts, controller
implementation patterns, and metrics labels.

Core Architectural Changes
  - Controller Terminology and Semantics
  - Verdict Model Transformation
  - Policy Engine Enhancements
Metrics Changes
  - phase label values: authorization → match
  - metric name: envoy_authz_controller_phase_duration_seconds → envoy_authz_controller_duration_seconds
